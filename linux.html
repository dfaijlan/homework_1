Linux

Linux is a Unix-like computer operating system that is considered
to be under the model of free and open-source software development
and distribution. The defining component of Linux is the Linux
kernel, which was released 25 years ago on September 17, 1991
by Linus Torvalds. While trailing behind Windows and MacOS as the
third most used Operating System, it is considered to be the leading
operating system used on servers and other big iron systems such
as mainframe computers. The underlying source code of Linux
may be used, modified, and distributed, whether it be commercially
or non-commercially.
