macOS

MacOS, previously known as Mac OS X and later OS X, is a family
of Unix-based graphical operating systems developed and maintained
by Apple Inc. It is designed to run on Apple's brand of personal
computers called a Macintosh, or Mac for short. It is currently
the second most used operating system after Microsoft Windows.
MacOS originally launched in 2001, which superseded the "classic"
Mac Os, which launched in 1984. This OS has been preinstalled
on all Macs since 2002.
