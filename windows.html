Microsft Windows

Microsoft Windows, or simply Windows, is a family of graphical
operating system that was developed and sold by the company Microsoft.
Windows is designed to run on different Microsoft platforms,
such as Windows personal computers and video game consoles
It was initially released on November 20, 1985; 31 years ago,
with the name Windows 1.0. Since then Windows has became the
dominant force in the personal computer market with over 90%
market share, surpassing MacOs, which was released a year prior.
The most recent release is Windows 10, which was released in
September 2016.
